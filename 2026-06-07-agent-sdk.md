# Agent SDK

The Claude Agent SDK lets you build custom multi-step AI agents that can use tools, call other agents, and maintain state across complex workflows — all programmatically from Python or TypeScript. Instead of a single prompt-response cycle, you define an agent that can reason, take actions, and loop until a goal is reached. This is the foundation for building autonomous coding assistants, research pipelines, and orchestrated multi-agent systems.

## Example

```python
import anthropic

client = anthropic.Anthropic()

tools = [
    {
        "name": "read_file",
        "description": "Read the contents of a file",
        "input_schema": {
            "type": "object",
            "properties": {
                "path": {"type": "string", "description": "File path to read"}
            },
            "required": ["path"]
        }
    }
]

messages = [{"role": "user", "content": "What is in ./README.md?"}]

# Agentic loop
while True:
    response = client.messages.create(
        model="claude-opus-4-8",
        max_tokens=1024,
        tools=tools,
        messages=messages
    )

    if response.stop_reason == "end_turn":
        print(response.content[0].text)
        break

    # Handle tool use
    if response.stop_reason == "tool_use":
        tool_call = next(b for b in response.content if b.type == "tool_use")
        # ... execute tool_call.name with tool_call.input ...
        # append assistant message + tool result to messages and loop
        break
```

## Try it yourself

Clone the [Anthropic quickstarts repo](https://github.com/anthropics/anthropic-quickstarts) and run the `computer-use-demo` or `customer-support-agent` example. Swap in `claude-sonnet-4-6` as the model, add a custom tool (e.g. one that calls a local API), and watch the agent decide when to invoke it.
